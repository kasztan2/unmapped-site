Welcome to the OSM Unmapped Project!

The idea

OSM Unmapped is a tool to find POIs potentially missing on OSM. It is comparing data from retail chains' websites with data from OSM and catching objects that are far from any object of that brand on OSM.

How to use data

In the Get data section there are links to geojson data on every brand that is currently supported.

Technicalities

Things used in this project:

    • Overpass API
